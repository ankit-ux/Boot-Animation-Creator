# Boot Animation Creator v2.5.0

Boot Animation Creator is a desktop application for creating Android boot animations from videos with a simple and modern workflow.

(Android/Android TV)

## Features

* Convert videos directly into Android `bootanimation.zip`
* Automatic frame extraction using FFmpeg
* Generates properly formatted `desc.txt`
* Creates Android-compatible uncompressed ZIP archives
* USB ADB support for direct installation
* Wireless ADB support
* Custom resolution presets
* Custom FPS configuration
* Preview generated frames before export
* Modern Electron-based interface
* Windows x64 support
